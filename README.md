\# Telco Customer Churn Prediction



\## Overview

This project uses a Random Forest Classifier to predict whether a customer will leave a telecommunications company based on their usage patterns and demographics.



\## How to use

1\. Install dependencies: `pip install pandas scikit-learn`

2\. Run the model: `python churn\_model.py`



\## Results

The model achieves approximately 79-80% accuracy in predicting customer behavior.
